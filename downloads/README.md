## Why we don't mirror the PDF in this repo

Three reasons:

1. **Size discipline.** GitHub is for source and discussion; the Archive is for permanence.
2. **Single source of truth.** When the author releases a corrected edition, only one canonical file updates.
3. **Archival guarantees.** Internet Archive provides preservation commitments that GitHub does not.

That said, **the EPUB and PDF are small enough that some users may prefer a repo-local copy.** If we add them later, they will live in this folder with a `SHA256SUMS` file for integrity verification.
